0fAvailable boot methods:07

0flive07, 0flive-gui07, or  0flive-serial07
  Start the live system with respective console.
  (default is 0flive07 with both VGA and serial consoles)
0finstall07, 0finstall-gui07, or 0finstall-serial07
  Start the installation.
0fexpert07,  0fexpert-gui07, and 0fexpert-serial07
  Start the installation in expert mode, for maximum control.
0frescue07,  0frescue-gui07, and 0frescue-serial07
  Start the installation in rescue mode, for safe recovery.
0fauto07,  0fauto-gui07, and 0fauto-serial07
  Start the installation in auto mode, for minimal admin input.

To use one of these boot methods, type it at the prompt, optionally
followed by any boot parameters. For example:
  boot: install acpi=off

If unsure, you should use the default boot method, with no special
parameters, by simply pressing enter at the boot prompt.

Except in expert mode, non-critical kernel boot messages are suppressed.
